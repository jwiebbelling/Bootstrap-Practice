About CSS and this class

A CSS-Framework is basically a set of CSS elements that were already made. Think of it as presets or templates! The whole thing was done beforehands so that other developers don't have to do it all from scratch.

A CSS-Preprocessor will be your best friend when it comes to writing CSS. When using CSS-Preprocessors, the repetitive tasks become almost fully automated and the number of bugs and errors will be significantly reduced. Let's use Sass as an example: it makes writing the code a lot easier and more dynamic, as long as other features.

We've learned about Bootstrap, making grids (and how they work), building websites, responsiveness and other HTML and CSS elements.
